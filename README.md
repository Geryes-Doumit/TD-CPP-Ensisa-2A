# TD C++ 2A IR ENSISA

To compile and run the C++ code, either go to the right folder using `cd` and run these commands :

```bash
g++ -o main.o *.cpp
./main.o
```

or directly run :

```bash
g++ -o main.o {path}/*.cpp
./main.o
```

with `{path}` being the path to the right folder, for example : `TD1/Exo1`.
